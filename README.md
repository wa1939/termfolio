# Notion-terminal

CLI-first personal website built with Next.js App Router, a Notion-backed journal, and a terminal-inspired interface with calmer editorial reading surfaces.

## Documentation

Start here if you are onboarding to the repo:

- `docs/AI_CONTEXT.md` - project overview, source-of-truth files, and active architecture
- `docs/DESIGN_HANDOFF.md` - visual intent and iteration brief for future UI work
- `docs/ROUTES.md` - route-by-route behavior and responsibilities
- `docs/INTEGRATIONS.md` - Notion, comments, env vars, and runtime fallbacks
- `docs/THEME_AND_STYLING.md` - active design language, typography, and styling guidance
- `docs/ACTIVE_VS_LEGACY.md` - active files versus likely legacy code
